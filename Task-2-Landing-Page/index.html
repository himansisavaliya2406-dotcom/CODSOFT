The Rent Attire

A full-stack MERN web application built to provide an online rental platform for premium wedding and traditional clothing. The system allows users to browse collections, check outfit availability for selected dates, place rental bookings, make payments, and track their orders seamlessly. The project emphasizes server-side validation, secure database handling, and a user-friendly interface.
